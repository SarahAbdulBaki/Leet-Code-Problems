  <!-- Title and Header Badges -->
  <h1>1. Two Sum <span style="float: right; font-size: 14px; font-weight: normal; color: #f5c242;"></span></h1>
  
  

  <!-- Problem Statement -->
  <p>Given an array of integers <code style="background-color: #f7f7f7; padding: 2px 4px; border-radius: 4px;">nums</code> and an integer <code style="background-color: #f7f7f7; padding: 2px 4px; border-radius: 4px;">target</code>, return <em>indices of the two numbers such that they add up to <code style="background-color: #f7f7f7; padding: 2px 4px; border-radius: 4px;">target</code></em>.</p>
  
  <p>You may assume that each input would have <strong><em>exactly</em> one solution</strong>, and you may not use the <em>same</em> element twice.</p>
  
  <p>You can return the answer in any order.</p>

  <p>&nbsp;</p>

  <!-- Examples -->
  <p><strong>Example 1:</strong></p>
  <pre style="background-color: #f8f8f8; border: 1px solid #ddd; padding: 10px; border-radius: 5px; font-family: monospace;">
<strong>Input:</strong> nums = [2,7,11,15], target = 9
<strong>Output:</strong> [0,1]
<strong>Explanation:</strong> Because nums[0] + nums[1] == 9, we return [0, 1].</pre>

  <p><strong>Example 2:</strong></p>
  <pre style="background-color: #f8f8f8; border: 1px solid #ddd; padding: 10px; border-radius: 5px; font-family: monospace;">
<strong>Input:</strong> nums = [3,2,4], target = 6
<strong>Output:</strong> [1,2]</pre>

  <p><strong>Example 3:</strong></p>
  <pre style="background-color: #f8f8f8; border: 1px solid #ddd; padding: 10px; border-radius: 5px; font-family: monospace;">
<strong>Input:</strong> nums = [3,3], target = 6
<strong>Output:</strong> [0,1]</pre>

  <p>&nbsp;</p>

  <!-- Constraints -->
  <p><strong>Constraints:</strong></p>
  <ul>
    <li><code style="background-color: #f7f7f7; padding: 2px 4px; border-radius: 4px;">2 &lt;= nums.length &lt;= 10<sup>4</sup></code></li>
    <li><code style="background-color: #f7f7f7; padding: 2px 4px; border-radius: 4px;">-10<sup>9</sup> &lt;= nums[i] &lt;= 10<sup>9</sup></code></li>
    <li><code style="background-color: #f7f7f7; padding: 2px 4px; border-radius: 4px;">-10<sup>9</sup> &lt;= target &lt;= 10<sup>9</sup></code></li>
    <li><strong>Only one valid answer exists.</strong></li>
  </ul>

  <p>&nbsp;</p>
  <p><strong>Follow-up:</strong> Can you come up with an algorithm that is less than <code style="background-color: #f7f7f7; padding: 2px 4px; border-radius: 4px;">O(n<sup>2</sup>)</code> time complexity?</p>

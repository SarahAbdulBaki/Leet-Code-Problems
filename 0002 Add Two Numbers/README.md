  <!-- Title and Header Badges -->
  <h1>2. Add Two Numbers <span style="float: right; font-size: 14px; font-weight: normal; color: #f5c242;"></span></h1>
  
  <p style="margin-top: 10px; margin-bottom: 20px;">
    <span style="background-color: #ffc01e; color: white; padding: 4px 10px; border-radius: 12px; font-size: 12px; font-weight: 500; margin-right: 5px;">Medium</span>
  </p>

  <!-- Problem Statement -->
  <p>You are given two <strong>non-empty</strong> linked lists representing two non-negative integers. The digits are stored in <strong>reverse order</strong>, and each of their nodes contains a single digit. Add the two numbers and return the sum as a linked list.</p>
  
  <p>You may assume the two numbers do not contain any leading zero, except the number 0 itself.</p>

  <p>&nbsp;</p>

  <!-- Examples -->
  <p><strong>Example 1:</strong></p>
  <pre style="background-color: #f8f8f8; border: 1px solid #ddd; padding: 10px; border-radius: 5px; font-family: monospace;">
<strong>Input:</strong> l1 = [2,4,3], l2 = [5,6,4]
<strong>Output:</strong> [7,0,8]
<strong>Explanation:</strong> 342 + 465 = 807.</pre>

  <p><strong>Example 2:</strong></p>
  <pre style="background-color: #f8f8f8; border: 1px solid #ddd; padding: 10px; border-radius: 5px; font-family: monospace;">
<strong>Input:</strong> l1 = [0], l2 = [0]
<strong>Output:</strong> [0]</pre>

  <p><strong>Example 3:</strong></p>
  <pre style="background-color: #f8f8f8; border: 1px solid #ddd; padding: 10px; border-radius: 5px; font-family: monospace;">
<strong>Input:</strong> l1 = [9,9,9,9,9,9,9], l2 = [9,9,9,9]
<strong>Output:</strong> [8,9,9,9,0,0,0,1]</pre>

  <p>&nbsp;</p>

  <!-- Constraints -->
  <p><strong>Constraints:</strong></p>
  <ul>
    <li>The number of nodes in each linked list is in the range <code style="background-color: #f7f7f7; padding: 2px 4px; border-radius: 4px;">[1, 100]</code>.</li>
    <li><code style="background-color: #f7f7f7; padding: 2px 4px; border-radius: 4px;">0 &lt;= Node.val &lt;= 9</code>.</li>
    <li>It is guaranteed that the list represents a number that does not have leading zeros.</li>
  </ul>
